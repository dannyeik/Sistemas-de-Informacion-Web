Enlace con el archivo CSS externo utilizando la etiqueta "link" dentro de la sección "head".

En el archivo HTML, el párrafo (p) tiene un estilo inline con style="color: red;", lo que hace que el texto del párrafo sea de color rojo. En el archivo CSS (index ejercicio 10 estilos.css), se define un estilo para el elemento "body" que cambia el color de fondo de la página a un tono azul claro (lightblue).
